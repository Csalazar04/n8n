# Automatización con n8n

## Flujo 1 — Registro de datos con notificación por email

### ¿Qué hace?
Recibe datos enviados por un formulario via Webhook, los procesa y los registra
automáticamente en Google Sheets. Dependiendo del resultado:
- ✅ **Éxito:** envía un email de confirmación con los datos recopilados
- ❌ **Error:** envía un email de alerta indicando que los datos no pudieron ser guardados

### Tecnologías usadas
- n8n (automatización)
- Google Sheets
- Gmail
- Webhook (HTTP)

### Aprendizajes
- Manejo de flujos con lógica de éxito/error
- Integración de APIs externas sin código
- Automatización de procesos de registro de datos
